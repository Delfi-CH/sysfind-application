# sysfind-application

application for managing operating system installers

[https://sysfind.app](https://sysfind.app)

## Build from source

1. Install the prerequisites [https://tauri.app/start/prerequisites/](https://tauri.app/start/prerequisites/)

2. Clone the repository `git clone https://github.com/Delfi-CH/sysfind-application.git`

3. Install frontend dependencies `npm install`

4. Build with `npm run tauri build`
